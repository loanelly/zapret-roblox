Закрытие
Драйвер продолжает работать как системная служба, даже после закрытия основной программы. Чтобы остановить службу, нужно: 

Нажать сочетание Win+X и выбрать «Windows Terminal (Администратор)» или «Командная строка (Администратор)».

В консоли последовательно выполнить команды: sc stop WinDivert, sc delete WinDivert. Если такие названия не подходят, можно использовать другие варианты: sc stop WinDivert14, sc delete WinDivert14 либо sc stop WinDivert64, sc delete WinDivert64.

В завершение обязательно перезапустить систему — только после перезагрузки Windows перестанет удерживать файл и позволит его удалить.
